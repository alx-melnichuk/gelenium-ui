The following attributes are supported 1: #

  • "id" - parameter for defining the unique identifier of the field (id="unique_identifier");
  • "config" - Defines a configuration object. More details.
  • "isDisabled" - a sign that the field is disabled (isDisabled);
  • "position" - Defines horizontal positioning. It can take the following values:
    • "start" - positioned at the left edge (default);
    • "center" - positioned in the middle;
    • "end" - positioned on the right edge;

Attributes are also supported 2: #

  • "isMaxWd" - set the maximum width when expanding the dropdown list of options.
    Without this setting, the maximum width of the options panel is determined by its content.
    The value of the minimum width of the options panel is always equal to the width of the parent.
  • css: '--glnac-min-width: 180px;' - defines the minimum width of the option list dropdown list of options.
  • css: '--glnac-max-width: 180px;' - defines the maximum width of the option list dropdown list of options.
  • "panelClass" - set the classes to be applied to the options panel. Supports the same syntax as 'ngClass'.

Attributes are also supported 3: #

  • "visibleSize" - sets the maximum number of visible option elements.
    The options bar is set to max-height = optionHeight * visibleSize.
    Where option height is optionHeight = lineHeight + 2 * 0.375 * fontSize;
  • css: '--glnac-max-height: 180px;' - defines the maximum height of the option list dropdown list of options.
  • "isNoAnimation" - hide the display of animation when expanding the dropdown list of options.
  • css: '--glnac-elevation: none;' - hide the display of shadow effects when expanding the dropdown list of options.

Attributes are also supported 4: #

  • "isClearOnEscape" - clear the input field when pressing 'Escape'.
  • "isNoCloseOnSelect" - the options panel will not close after an option is selected.
  • "isNoRipple" - Disables the ripple effect.
  • "isNoOpenOnMouse" - the options panel will not open/close on mouse click.
  • "isOpenOnFocus" - the options panel opens after receiving input focus.
  • "css: font-size: 1.125em;" - You can change the font size for GlnAutocomplete and that font size will be applied to the available options in the list.
