"Transitions" of snackbars #

The display of "transitions" of snackbars can be changed. The configuration parameter "transition" is responsible for the transition type.

And it can take the following values:
  • 'grow' - "scaling" transition (css - transform: scale(.3)) (default);
  • 'fade' - "manifestation" transition (css - opacity: 0);
  • 'blur' - "blur" transition (css - filter: blur(20px));
  • 'slide' - "slide up" transition (css - transform: translate(0px, 120vh));
  • 'slide-dw' - "slide down" transition (css - transform: translate(0px, -120vh));
  • 'slide-lf' - "slide left" transition (css - transform: translate(-120vw, 0px));
  • 'slide-rg' - "slide right" transition (css - transform: translate(120vw, 0px));
  • 'turn' - "3D turn along x-axis" transition (css - transform: perspective(400px) rotateX(90deg));
  • 'turn-y' - "3D turn along y-axis" transition (css - transform: perspective(400px) rotateY(90deg));

snackbar with action button #

The snackbar component provides the ability to display an action button.

    <button class="snat4-btn"
      tabindex="0"

      <span>Open snackbar with action</span>
    </button>
  </div>
</div>
 
import {
  Component, ViewEncapsulation
} from '@angular/core';
import { GlnSnackbarService } from 'gelenium-ui';

@Component({
  selector: 'app-snackbar',
  templateUrl: './snackbar.component.html',
  styleUrls: ['./snackbar.component.scss'],
  encapsulation: ViewEncapsulation.None,
})
export class SnackbarComponent {
  constructor(private snackbarService: GlnSnackbarService) {}

  public clickAction(
    message: string, action: string, config: any
  ): void {
    this.snackbarService.open(message, action, config)
    .result.then((response) => {
      // response=undefined - When closing by the "close" button.
      // response=UNDO      - When closing by the "action" button.
      console.log(`simple action response=${response}`);
    });
  }
