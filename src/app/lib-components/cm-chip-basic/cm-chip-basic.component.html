Basic. #

To use this component, in your module you need to add: 
import { GlnChipModule } from 'gelenium-ui';

The component is designed to display the name of an entity in the form of a small panel. Also, this panel may optionally have additional attributes (icons or buttons).

It comes in two versions:
  • with an outline (exterior="outlined")
  • with a fill (exterior="filled")
For example: 
<gln-chip label="chip"></gln-chip>
{{ labelShowSource }}
