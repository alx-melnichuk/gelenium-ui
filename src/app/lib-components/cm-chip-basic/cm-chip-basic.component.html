Basic. #

To use this component, in your module you need to add: 
import { GlnChipModule } from 'gelenium-ui';

The component is designed to display the name of an entity in the form of a small panel. Also, this panel may optionally have additional attributes (icons or buttons).

It comes in two versions:
  • with an outline (exterior="outlined")
  • with a fill (exterior="filled")
For example: 
<gln-chip label="chip"></gln-chip>
_

Attributes. #

The following attributes are supported:
  • "config" - Defines a configuration object. More details.
  • "exterior" - Determines the appearance of an element.
    It can take the following values:
    • "outlined";
    • "filled";
  • "isDisabled" - a sign that the field is disabled (isDisabled);
  • "isElevation" - a sign displaying the height of the element (isElevation);
  • "isNoRipple" - Disables the ripple effect.

Size #

The "size" property determines the size of the element (its height).

It takes the following values:
  • "short" - sets the height to 24px;
  • "little" - sets the height to 28px;
  • "small" - sets the height to 32px (default);
  • "middle" - sets the height to 36px;
  • "wide" - sets the height to 40px;

The height of the "delete" button is determined by the formula: Math.round(size * 0.672).

It is possible to specify the height of an element. To do this, in the "config.size" property, we define the required height in pixels.

Ornaments. #

For the GlnChip component, it is possible to display an ornament (picture or text).
To display the ornament on the left, you need to add a block with the "gln-orn-lf" attribute inside the "gln-chip".
To display the ornament on the right, you need to add a block with the "gln-orn-rg" attribute inside the "gln-chip".

Example: 
<gln-chip>
  <div gln-orn-lf><span>left</span></div>
  <div gln-orn-rg><span>right</span></div>
</gln-chip>

For blocks: <span>left</span> and <span>right</span> you need to specify the value of horizontal indents.

Ornament - text. #

The css class "chor-text" defines the horizontal padding.
The css class "chor-fn-sz" defines the font size as 13px.

Config #

If your application will have a single style for GlnChip elements, then you can set a single configuration. To do this, you need to create an object with the GlnChipConfig interface and define the required values. Next, specify the resulting object in the list of providers for the entire application (at the level of the AppComponent). 

const glnChipConfigDefault: GlnChipConfig = {
  exterior: 'filled',
  isElevation: true,
  size: 'middle',
};

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.scss'],
  providers: [
    { provide: GLN_CHIP_CONFIG, useValue: glnChipConfigDefault }
  ],
})
export class AppComponent {
  title = 'demo';
}

After that, all "GlnChip" components will have the same configuration.

1. Configuration with exterior="filled", isElevation=true, size="middle". #

For example, let's prepare a configuration in which:

  • "exterior" will have the value "filled";
  • "isElevation" will have the value "true";
  • "size" will have the value "middle"
const glnChipConfigDefault: GlnChipConfig = {
  exterior: 'filled',
  isElevation: true,
  size="middle",
};

2. Configuration with exterior="outlined", isElevation=false, size="small". #

Suppose we have a provider in our application with "GlnChipConfig" and all "chip" elements accept its default values. We can change these values for an individual element. To do this, you need to specify in this element those properties that do not match the provider's configuration.

For example, let's specify the following parameters: 

exterior="outlined",
isElevation=false,
size="small",

In other words, there is the following order of determining the configuration parameters:

  • Standard values are taken. 
    exterior="outlined",
isElevation=false,
isNoRipple=false,
size="small",
  • Values from the default configuration provider (if there is one).
  • Values from input parameters (if there is one).
1. Default config: exterior="filled", isElevation=true, size="middle".
2. exterior="outlined", isElevation=false, size="small".
