Palette. #

The GlnChip component has the ability to change its color.

For example: 
<gln-chip
  [glnColor]="{ default: 'primary' }"
  formControlName="model02a">
  Label
</gln-chip>

This means that you want to use the system color "primary" as the "default" color.

To use the glnColor directive, you need to add in your module: 
import { GlnColorModule } from 'gelenium-ui';
