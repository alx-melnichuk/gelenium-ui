Attributes. #

The following attributes are supported: #

  • "id" - parameter for defining the unique identifier of the field (id="unique_identifier");
  • "config" - Defines a configuration object. More details.
  • "exterior" - Determines the appearance of an element.
    It can take the following values:
    • "outlined";
    • "underline";
    • "standard";
  • "helperText" - the parameter is used to provide context for a given field or error message (helperText="A value must be entered.");
  • "isDisabled" - a sign that the field is disabled (isDisabled);
  • "isError" - a sign that the field is in error (isError);
  • "isLabelShrink" - An indication that the label has moved up and shrunk.
  • "isPlaceholder" - specifies that the value of the "label" field be displayed as the "placeholder" attribute for the "input" element (isPlaceholder);
  • "isReadOnly" - a sign that the field is read-only (isReadOnly);
  • "isRequired" - a sign that the field is required (isRequired);
  • "label" - The content of the label.

Attributes are also supported: #

  • "wdFull" - an attribute sets the width to the width of the parent (width: 100%).
    Without this parameter, the width is determined by the content.
    The default minimum width is frameSizeValue * 1.1.
    The content of "helperText" also affects the width of the component.
    The value of the minimum width can be specified by the css citile or by setting the css variable "--glnsl-min-width".
  • "position" - Defines horizontal positioning. It can take the following values:
    • "start" - positioned at the left edge;
    • "center" - positioned in the middle;
    • "end" - positioned on the right edge;
  • "visibleSize" - Specifies the maximum number of visible option elements. This value is multiplied by the height of the option and defines the maximum height of the options panel.
  • "isNoIcon" - Hide the display of the status icon of the dropdown list of options.
  • "isNoAnimation" - Hide the display of the animation when the dropdown list of options is expanded.
