Basic. #

To use this component, in your module you need to add: 
import { GlnSnackbarModule } from 'gelenium-ui';

The GlnSnackbar component is designed to display brief notifications. It is also called "toast".
The component informs the user about some event: data calculation process started, data saving completed successfully (or not) and so on.
By default, a short message temporarily appears at the bottom of the screen. In this case, the input focus is not transferred to it. The notification panel can contain a button associated with this action.

For example: 
<button type="button" (click)="onClick()">
  <span>Open notification</span>
</button>

import { GlnSnackbarModule } from 'gelenium-ui';
@Component({ ... })
export class NotificationComponent {
  constructor(private snackbarService: GlnSnackbarService) {}
  public onClick(): void {
    this.snackbarService.open('text notification !');
  }
}
