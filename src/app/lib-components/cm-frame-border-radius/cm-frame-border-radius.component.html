Border radius. #

For components that use GlnFrame it is possible to change the border-radius css property. This requires creating a css variable "--glnfr-br-rd" at the level of the component or any parent element. The value of this css variable can be any valid (as described by "border-radius" ) value.

For example: 
<div>
  <gln-input style="--glnfr-br-rd: 25px; --glnfr-pd-lf: 29px;"
    formControlName="model05"
    label="Enter value"
    exterior="outlined">
  </gln-input>
</div>
 Another example: 
<div style="--glnfr-br-rd: 25px; --glnfr-pd-lf: 29px;">
  <gln-input formControlName="model05"
    label="Enter value"
    exterior="outlined">
  </gln-input>
</div>

In the same way, you can change the border-radius for the "Underline" mode. 

<div>
  <gln-input style="--glnfr-br-rd: 25px 25px 0 0; --glnfr-pd-lf: 20px;"
    formControlName="model05"
    label="Enter value"
    exterior="underline">
  </gln-input>
</div>

This value for the fillet radius corresponds to 50% of the height of the current element.
The minimum value for the fillet radius is 0.
The maximum value for the fillet radius is 50% of the height of the current element.

You can also change the value of the left / right indents, for a more convenient perception.
To change the left (and right) indent of the label, you need to add the css variable '--glnfr-pd-lf'.

import {
  Component, ViewEncapsulation
} from '@angular/core';
import {
  FormControl, FormGroup
} from '@angular/forms';

@Component({
  selector: 'app-frame',
  templateUrl: './frame.component.html',
  styleUrls: ['./frame.component.scss'],
  encapsulation: ViewEncapsulation.None,
})
export class FrameComponent {
  public borderRadius = 50;
  public ratioOutlined = 0.58;
  public ratioUnderline = 0.4;

  public exterior05 = 'outlined';
  public isAddPadding05 = true;
  public hideLabel05 = false;

  public formGroup05: FormGroup = new FormGroup({
    model05a: new FormControl('Demo Size Short', []),
    model05b: new FormControl('Demo Size Small', []),
    model05c: new FormControl('Demo Size Middle', []),
    model05d: new FormControl('Demo Size Wide', []),
    model05e: new FormControl('Demo Size Large', []),
    model05f: new FormControl('Demo Size Huge', []),
  });

  public getValue(frameSize: string | null): number {
    return GlnFrameSizeUtil.getValue(
      GlnFrameSizeUtil.convert(frameSize)) ?? 0;
  }
  public getRadius(value: number, exterior: string): string {
    return exterior === 'underline'
      ? value + 'px ' + value + 'px 0 0' : value + 'px';
  }
  public getRatio(exterior: string): number {
    return exterior === 'underline'
      ? this.ratioUnderline : this.ratioOutlined;
  }
}
