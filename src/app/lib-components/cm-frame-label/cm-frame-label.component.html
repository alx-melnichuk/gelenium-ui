Label #

The "label" property defines the contents of the legend for the current component. If the component has no value set, then the legend is displayed as a placeholder. When the user enters a value for such a component, the legend shrinks and moves up. This is the default behavior for a legend.

Sometimes you want to deviate from the standard legend behavior. For example, you want to display the legend at the top only. This requires adding the "isLabelShrink" attribute.

Another example, if you require a component without a legend, then simply leave out the "label" parameter.

exterior: {{ labelOutlined }} {{ labelUnderline }} {{ labelStandard }}
Attribute: isLabelShrink.
Attribute: without Label.

  FormControl, FormGroup

  styleUrls: ['./frame.component.scss'],
  encapsulation: ViewEncapsulation.None,
})

  public exterior03a = 'outlined';
  public formGroup03a: FormGroup = new FormGroup({
    model03a: new FormControl('', []),
    model03b: new FormControl('Hello World', []),
  });
}
