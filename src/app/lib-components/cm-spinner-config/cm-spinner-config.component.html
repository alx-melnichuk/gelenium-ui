Config #

If your application will have a single style for GlnSpinner elements, then you can set a single configuration. To do this, you need to create an object with the GlnSpinnerConfig interface and define the required values. Next, specify the resulting object in the list of providers for the entire application (at the level of the AppComponent). 

const glnSpinnerConfigDefault: GlnSpinnerConfig = {
  exterior: 'underline',
  size: 'wide',
};

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.scss'],
  providers: [
    { provide: GLN_SPINNER_CONFIG, useValue: glnSpinnerConfigDefault }
  ],
})
export class AppComponent {
  title = 'demo';
}

After that, all "GlnSpinner" components will have the same configuration.

GlnSpinnerConfig interface. #

Name Description
isExternal?: boolean
  | undefined;		 Displays the specified content instead of an inline SVG image.
(default = false;)
isNoAnimation?: boolean
  | undefined;		 Disable animation execution.
(default = false;)
isNoPulsate?: boolean
  | undefined;		 Disable circle pulsate.
(default = false;)
progress?: number
  | undefined; 		The value of the filled circle arc value from 0 to 1.
(default 0.7).
size?: number
  | string
  | undefined;		 Determines the size of the element.
It can take the following values:
  • "short" - height: 38px;
  • "small" - height: 44px (default);
  • "middle" - height: 50px;
  • "wide" - height: 56px;
  • "large" - height: 62px;
  • "huge" - height: 68px;
This property can also contain the element's height in pixels, either as a string or as a number.
The value in the units of "pixels" allow you to accurately calculate the size of internal elements.
For example: size="57" or [size]="57"
strokeWd?: number
  | undefined; 		The value specifies the stroke width.
(default = size / 10).

1. Configuration with "outlined" and "small". #

For example, let's prepare a configuration in which:

  • "size" will have the value "short" (height to 38px);
const glnSpinnerConfigDefault: GlnSpinnerConfig = {
  size: 'short',
};

2. Configuration with "middle". #

Suppose we have a provider in our application with "GlnSpinnerConfig" and all "GlnSpinner" elements accept its default values. We can change these values for an individual element. To do this, you need to specify in this element those properties that do not match the provider's configuration.

For example, let's specify the following parameters: 

size="middle"

In other words, there is the following order of determining the configuration parameters:

  • Standard values are taken. 
    isExternal=false,
isNoAnimation=false,
isNoPulsate=false,
progress=0.7,
size='small',
strokeWd=size/10,
  • Values from the default configuration provider (if there is one).
  • Values from input parameters (if there is one).
1. Default config: "short".
2. size="middle".

import {
  Component, ViewEncapsulation
} from '@angular/core';
import {
  GlnSpinnerConfig, GLN_SPINNER_CONFIG
} from 'gelenium-ui';

const glnSpinnerConfigDefault: GlnSpinnerConfig = {
  size: 'short',
};

@Component({
  selector: 'app-spinner',
  templateUrl: './spinner.component.html',
  styleUrls: ['./spinner.component.scss'],
  encapsulation: ViewEncapsulation.None,
  providers: [{ provide: GLN_SPINNER_CONFIG, useValue: glnSpinnerConfigDefault }],
})
export class SpinnerComponent {
  public isNoAnimation08a: boolean = true;
}
 
app-spinner {

  }
