API: GlnSnackbar. #

Selector: gln-snackbar

Exported as: glnSnackbar

Properties. #

Name Description
@Input() params: GlnSnackbarParams; Parameters to display the snackbar.
The GlnSnackbarParams interface contains:
  • message: string - short message text;
  • action?: string - action button name;
  • msgType?: string - message type name;
    Takes values:
    • 'default' - default value (or if not set);
    • 'error' - error type (color "danger");
    • 'success' - success type (color "success");
    • 'info' - information type (color "info");
    • 'warning' - warning type (color "warning");
  • isInvert?: boolean - sign of inversion of background color and text color;
  • isNoClose?: boolean - sign of hiding the close button (default - false);

Methods #

close (): void;
Close the snackbar.
action (): void;
Close the snackbar (by the specified action).
The "action" property must be specified.

Class GlnSnackbarContainerComponent. #

This component is a container for a list of snack bars with a single configuration.
If the configuration of two snackbars is the same, then they will have one common GlnSnackbarContainerComponent instance.
Otherwise, if two snackbars don't have the same configuration, they will have two different GlnSnackbarContainerComponent instances.

The unique key is determined by the following configuration properties:
  • horizontal;
  • overlayClasses
  • vertical

CSS variables for GlnSnackbarContainerComponent, optional. You can override the default values. #

Name Description
--glnsbc-mr: 1rem; The css size of the "margin" property for the current "wrapper" element.
--glnsbc-mr-mini: 0.5rem; The css size of the "margin" property for the current "wrapper" element when the screen size is less than 600px.
--glnsbc-trn-dur: 260ms; The runtime value (milliseconds) to animate the opening/closing of the snackbar element.
If you want to disable the animation, then you need to set the value of the parameter to zero. 
gln-snackbar-container {
  --glnsbc-trn-dur: 0;
