API: GlnButton #

Selector: gln-button

Exported as: glnButton

Properties #

Name Description
@Input() id: string; ID of the native button.
(default = `glnbt-${uniqueIdCounter++}`;)
@Input() config: GlnButtonConfig
  | null
  | undefined; 		Defines a configuration object.
More details.
@Input() exterior: string
  | null
  | undefined; 		Determines the appearance of an button.
It can take the following values:
  • "contained";
  • "outlined";
  • "text" (default);
@Input() frameSize: string
  | null
  | undefined; 		Determines the size of the button.
More details.
It can take the following values:
  • "short" - height: 38px;
  • "small" - height: 44px (default);
  • "middle" - height: 50px;
  • "wide" - height: 56px;
  • "large" - height: 62px;
  • "huge" - height: 68px;
@Input() isDisabled: string
  | boolean
  | null
  | undefined; 		The element is disabled.
Attribute (no content).
@Input() isNoRipple: string
  | boolean
  | null
  | undefined; 		Flag for disabling the ripple effect.
Attribute (no content).
@Input() ornamLfAlign: string
  | null
  | undefined; 		Specifies the height justification for the left ornament.
(default = 'default';) It can take the following values:
  • "default" - depending on frame type
  • "center" - in the center
  • "flex-start" - at the top
  • "flex-end" - at the bottom
  • "baseline" - along the baseline
  • "stretch" - at the top
@Input() ornamRgAlign: string
  | null
  | undefined; 		Specifies the height justification for the right ornament.
(default = 'default';) It can take the following values:
  • "default" - depending on frame type
  • "center" - in the center
  • "flex-start" - at the top
  • "flex-end" - at the bottom
  • "baseline" - along the baseline
  • "stretch" - at the top
