  • "flexEnd" - defines bottom alignment
  • "baseline" - defines baseline alignment
  • "stretch" - stretches across the entire height
@Input() size: number
  | string
  | null
  | undefined; 		Determines the size of the button.
More details.
It can take the following values:
  • "short" - height: 38px;
  • "small" - height: 44px (default);
  • "middle" - height: 50px;
  • "wide" - height: 56px;
  • "large" - height: 62px;
  • "huge" - height: 68px;
This property can also contain the element's height in pixels, either as a string or as a number.
The value in the units of "pixels" allow you to accurately calculate the size of internal elements.
For example: size="57" or [size]="57"
@Output() focused: EventEmitter<void>; Fires when the element receives input focus.
@Output() blured: EventEmitter<void>; Fires when an element loses input focus.
