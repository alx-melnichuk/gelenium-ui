Basic. #

To use this component, in your module you need to add: 
import { GlnAutocompleteModule, GlnInputModule } from 'gelenium-ui';

The GlnAutocomplete component allows you to supplement the input of the GlnInput component with a panel of suggested options.

For example: 
<gln-input formControlName="model01a"
  label="Enter value"
  [glnAutocompleteTrigger]="autocomplete01">
  </gln-input>
  <gln-autocomplete #autocomplete01="glnAutocomplete">
    <gln-option value="mango">Mango</gln-option>
    <gln-option value="lemon">Lemon</gln-option>
    <gln-option value="orange">Orange</gln-option>
    <gln-option value="kiwi">Kiwi</gln-option>
  <gln-autocomplete>

Attributes. #

The following attributes are supported: #

  • "id" - parameter for defining the unique identifier of the field (id="unique_identifier");
  • "config" - Defines a configuration object. More details.
  • "isDisabled" - a sign that the field is disabled (isDisabled);
  • "position" - Defines horizontal positioning. It can take the following values:
    • "start" - positioned at the left edge (default);
    • "center" - positioned in the middle;
    • "end" - positioned on the right edge;

Attributes are also supported: #

  • "isMaxWd" - set the maximum width when expanding the drop-down list of options.
  • "visibleSize" - sets the maximum number of visible option elements.
    The options bar is set to max-height = optionHeight * visibleSize.
    Where option height is optionHeight = lineHeight + 2 * 0.375 * fontSize;
  • "isNoAnimation" - hide the display of animation when expanding the drop-down list of options.
  • css: '--glnolp-elevation: none;' - hide the display of shadow effects when expanding the drop-down list of options.
