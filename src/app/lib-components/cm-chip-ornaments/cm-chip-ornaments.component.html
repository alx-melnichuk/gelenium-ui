Ornaments. #

For the GlnChip component, it is possible to display an ornament (picture or text).
To display the ornament on the left, you need to add a block with the "gln-orn-lf" attribute inside the "gln-chip".
To display the ornament on the right, you need to add a block with the "gln-orn-rg" attribute inside the "gln-chip".

Example: 
<gln-chip>
  <div gln-orn-lf><span>left</span></div>
  <div gln-orn-rg><span>right</span></div>
</gln-chip>

For blocks: <span>left</span> and <span>right</span> you need to specify the value of horizontal indents.

Ornament - button. #

A "button" element can be used as an ornament.
When a "button" element is clicked, a message is written to the console.
If an event is being processed on the "ornament" and you do not want the ripple effect to work, then you need to cancel the further propagation of this event ($event.stopPropagation();).

Section: "gln-orn-lf" + button.
Section: "gln-orn-rg" + button.
{{ labelShowSource }} 
<div class="app-group">

  <div class="app-elem">
    <div>
      <small><em>Section: "gln-orn-lf" + button.</em></small>
    </div>
    <gln-chip class="chor-fn-sz"
      exterior="outlined"
      label="Outlined orn-left (button)">
      <button gln-orn-lf
        title="delete"
        class="chor-btn chor-btn-bg"
        (click)="showMsg('Outlined orn-left (button)', $event)">
        <span class="chor-label">✕</span>
      </button>
    </gln-chip>
  </div>

  <div class="app-elem">
    <div>
      <small><em>Section: "gln-orn-rg" + button.</em></small>
    </div>
    <gln-chip class="chor-fn-sz"
      exterior="filled"
      label="Filled orn-right (button)">
      <button gln-orn-rg
        title="delete"
        class="chor-btn"
        (click)="showMsg('Filled orn-right (button)', $event)">
        <i class="chor-del"></i>
      </button>
    </gln-chip>
  </div>

</div>
