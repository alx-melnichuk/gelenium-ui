Config. #

If your application will have a single style for GlnInput elements, then you can set a single configuration. To do this, you need to create an object with the GlnInputConfig interface and define the required values. Next, specify the resulting object in the list of providers for the entire application (at the level of the AppComponent). 

const glnInputConfigDefault: GlnInputConfig = {
  exterior: 'outlined',
  frameSize: 'small',
};

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.scss'],
  providers: [
    { provide: GLN_INPUT_CONFIG, useValue: glnInputConfigDefault }
  ],
})
export class AppComponent {
  title = 'demo';
}

After that, all input elements based on the "GlnInput" component will have the same configuration.

1. Configuration with "outlined" and "small". #

For example, let's prepare a configuration in which:

  • "exterior" will have the value "outlined";
  • "frameSize" will have the value "small" (height to 44px);
const glnInputConfigDefault: GlnInputConfig = {
  exterior: 'outlined',
  frameSize: 'small',
};

2. Configuration with "underline" and "wide". #

Suppose we have a provider in our application with "GlnInputConfig" and all input elements accept its default values. We can change these values for an individual element. To do this, you need to specify in this element those properties that do not match the provider's configuration.

For example, let's specify the following parameters: 

frameSize="wide"
exterior="underline"
isRequired

In other words, there is the following order of determining the configuration parameters:

  • Standard values are taken. 
    exterior="outlined", // GlnFrameConfig
frameSize="middle",  // GlnFrameConfig
isLabelShrink=false, // GlnFrameConfig
isNoAnimation=false, // GlnFrameConfig
isNoLabel=false,     // GlnFrameConfig
isRequired=false,    // GlnFrameConfig

isError=false,
isReadOnly=false,
ornamLfAlign='default',
ornamRgAlign='default',
  • Values from the default configuration provider (if there is one).
  • Values from input parameters (if there is one).

import {
  Component, ViewEncapsulation
} from '@angular/core';
import {
  FormControl, FormGroup
} from '@angular/forms';
import {
  GlnInputConfig, GLN_INPUT_CONFIG
} from 'gelenium-ui';

const glnInputConfigDefault: GlnInputConfig = {
  exterior: 'outlined',
  frameSize: 'small',
};

@Component({
  selector: 'app-input',
  templateUrl: './input.component.html',
  styleUrls: ['./input.component.scss'],
  encapsulation: ViewEncapsulation.None,
  providers: [{
    provide: GLN_INPUT_CONFIG, useValue: glnInputConfigDefault
  }],
})
export class InputComponent {
  public formGroup07: FormGroup = new FormGroup({
    model07a: new FormControl('', []),
    model07b: new FormControl('Hello World', []),
  });
}
 
app-input {

    display: flex;
    flex-wrap: wrap;
  }
  .app-elem {
    min-width: 26ch;
    margin: 10px 10px;
  }
}
