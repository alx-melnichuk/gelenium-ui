Attributes

The following attributes are supported:

The following attributes are supported:

  • "noUppercase" is an attribute that cancels the display of text in uppercase;
  • "isDisabled" - a sign that the field is disabled (isDisabled);
  • "wdFull" is an attribute sets the width to the width of the parent (width: 100%);
  • "noElevation" is an attribute to remove button elevation (only for "contained");
