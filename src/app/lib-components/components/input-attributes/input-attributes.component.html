Attributes

The following attributes are supported:

  • "id" - parameter for defining the unique identifier of the input field (id="unique_identifier");
  • "config" - Defines a configuration object. More details.
  • "exterior" - Determines the appearance of an element.
    It can take the following values:
    • "outlined";
    • "underline";
    • "standard";
  • "helperText" - the parameter is used to provide context about the input of the field or error message (helperText="Some important text").
  • "hoverColor" - When hovering the mouse, the label and frame will have the same color as when they received focus.
  • "isDisabled" - a sign that the field is disabled (isDisabled);
  • "isError" - a sign that the field is in error status (isError);
  • "isReadOnly" - read-only access flag (isReadOnly);
  • "isRequired" - a sign that the field is required (isRequired);
  • "label" - The content of the label.
  • "lbShrink" - An indication that the label has moved up and shrunk.
  • "noLabel" - Hide label display.
  • "type" - indicate the type of the input element (type="text");
  • "autoComplete" - the value of the autocomplete parameter for the input field.
      autoComplete="off" - The browser is not permitted to automatically enter or select a value for this field.
      autoComplete="new-password" - A new password.
      autoComplete="current-password" - The user's current password.
  • "wdFull" - an attribute sets the width to the width of the parent (width: 100%).

The "type" attribute can take the following values:

  • 'color';
  • 'date';
  • 'datetime-local';
  • 'email';
  • 'month';
  • 'number';
  • 'password';
  • 'search';
  • 'tel';
  • 'text';
  • 'time';
  • 'url';
  • 'week';

For a field with the "color" type, it is desirable to set a non-empty value (for example, "#000000"), otherwise, a warning may appear:
platform-browser.js:745 The specified value "" does not conform to the required format. The format is "#rrggbb" where rr, gg, bb are two-digit hexadecimal numbers.
This warning appears when there are two input elements of type "color" and "date".
