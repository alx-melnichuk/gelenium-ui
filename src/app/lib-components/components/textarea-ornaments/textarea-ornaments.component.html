Ornaments for the textarea.

It is possible to add a different ornament (picture or text) for the input line. It will appear as a prefix or suffix for the input string. You can define the reaction to pressing an element of the ornament.
To add a prefix, you need to add a tag with the "grnOrnamentLf" attribute inside the "grn-input" tag. (<span grnOrnamentLf>kg</span>)
To add a suffix, you need to add a tag with the "grnOrnamentRg" attribute inside the "grn-input" tag. (<span grnOrnamentRg>cm</span>)
Adding a prefix or suffix does not disable legend animation for that input field.
