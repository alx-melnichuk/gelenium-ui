Ornaments for the "button" element.

For the GrnButton component, it is possible to display an ornament (picture or text). It will appear as a prefix or suffix for the button name.

exterior: {{ labelContained }} {{ labelOutlined }} {{ labelText }}
Section: "grnOrnamentLf" + text.
+ Add new value
Section: "grnOrnamentRg" + text.
~ Edit current value
