Label

The "label" property defines the contents of the legend for the current component. If the component has no value set, then the legend is displayed as a placeholder. When the user enters a value for such a component, the legend shrinks and moves up. This is the default behavior for a legend.

Sometimes you want to deviate from the standard legend behavior. For example, you want to display the legend at the top only. This requires adding the "lbShrink" attribute.

Another example, you need to hide the legend display. This requires adding the "noLabel" attribute.
