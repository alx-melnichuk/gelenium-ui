Capability

Since this is a textarea field, when you click on "Enter", a new line is added. The number of visible lines depends on the height of the element. The rest of the lines can be seen by scrolling.
There are the following parameters that control the height of the element:

  • "cntRows" - sets the number of visible rows;
    The height of the text area is calculated as "cntRows" multiplied by the height of the row.
  • "minRows" - sets the minimum number of visible rows;
    The minimum height of the text area is calculated as "minRows" multiplied by to the line height. Active if "cntRows" is not specified.
  • "maxRows" - sets the maximum number of visible rows;
    The maximum height of the text area is calculated as "maxRows" multiplied by to the line height. Active if "cntRows" is not specified.
  • If none of "cntRows", "maxRows", "maxRows" is specified then the height of the text area is calculated as "number of lines" multiplied by the height of the line.

import {
  Component, ViewEncapsulation
} from '@angular/core';
import {
  FormControl, FormGroup
} from '@angular/forms';

@Component({
  selector: 'app-textarea',
  templateUrl: './textarea.component.html',
  styleUrls: ['./textarea.component.scss'],
  encapsulation: ViewEncapsulation.None,
})
export class TextareaComponent {
  public text1 = 'line 1\nline 2';
  public text2 = this.text1 + '\nline 3\nline 4\nline 5';
  public formGroup04: FormGroup = new FormGroup({
    model04a: new FormControl(this.text1, []),
    model04b: new FormControl(this.text2, []),
    model04c: new FormControl(this.text1, []),
    model04d: new FormControl(this.text2, []),
  });
  public exterior04 = 'outlined';
}
