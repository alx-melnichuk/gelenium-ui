Validation

The "isError" parameter sets the field to the error display state. The "helperText" parameter can be used to display an error message.

In case the field is in error:

  • displays a red box for "outlined";
  • red underline is displayed for "underline" or "standard";
  • the content of "label" is displayed in red;
  • the content of "helperText" is displayed in red;
  • the specified elements are displayed in red even if the field is disabled (isDisabled);

An error condition also occurs when the constraints set by the reactive form and attributes are violated.

import {
  Component, ViewEncapsulation
} from '@angular/core';
import {
  FormControl, FormGroup, Validators
} from '@angular/forms';

@Component({
  selector: 'app-input',
  templateUrl: './input.component.html',
  styleUrls: ['./input.component.scss'],
  encapsulation: ViewEncapsulation.None,
})
export class InputComponent {
  public minLength03 = 4;
  public controls03 = {
    input03a: new FormControl('', []),
    input03b: new FormControl('', [
      Validators.required,
      Validators.minLength(this.minLength03)
    ]),
    input03c: new FormControl('Hello World', []),
  };
  public formGroup03: FormGroup = new FormGroup(this.controls03);
  public exterior03 = 'outlined';
}
