Basic

The GrnButton component is a styled button.
It comes in three versions:

  • as a container (exterior="contained")
  • as an outline (exterior="outlined")
  • as text (exterior="text" - default)

The content of the grnButton component is determined by the content between its opening and closing tags. This can be a fragment of html elements that will be displayed as the content of the button.
Another option, the content of the hyperlink - in this case, the hyperlink will remain. In practice, there are often situations when a button is displayed on the design, but in terms of functionality it is a hyperlink. The second option is designed for such situations.

import {
  Component, ViewEncapsulation
} from '@angular/core';

@Component({
  selector: 'app-button',
  templateUrl: './button.component.html',
  styleUrls: ['./button.component.scss'],
  encapsulation: ViewEncapsulation.None,
})
export class ButtonComponent {
}
 
.b-items {
  display: flex;
  flex-wrap: wrap;
  &>:first-child {
    margin-left: 3%;
  }
  &>:not(first-child) {
    margin-left: 7%;
  }
}

Button with "contained" exterior.

Buttons with the "contained" exterior are for important primary actions.

Primary Primary Disabled Link Link Disabled
{{ labelShowSource }} 
<div class="b-items">

  <grn-button exterior="contained">
    <span>Primary</span>
  </grn-button>

  <grn-button exterior="contained"
    isDisabled>
    <span>Primary Disabled</span>
  </grn-button>

  <grn-button exterior="contained">
    <a [routerLink]="[routerLink]"
      [fragment]="fragment1">
      Link
    </a>
  </grn-button>

  <grn-button exterior="contained"
    isDisabled>
    <a [routerLink]="[routerLink]"
      [fragment]="fragment1">
      Link Disabled
    </a>
  </grn-button>

</div>
