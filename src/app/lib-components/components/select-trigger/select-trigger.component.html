Trigger #

It is possible to independently customize the display layout of the selected options. This requires the use of "GlnSelectTrigger".

For example: 
<gln-select formControlName="model05"
  #selectModel05="glnSelect"
  label="Enter value"
  wdFull>
  <gln-select-trigger>
    {{ selectModel05.value }}
  </gln-select-trigger>
  <gln-option><em>None</em></gln-option>
  <gln-option value="mango">Mango</gln-option>
  <gln-option value="lemon">Lemon</gln-option>
  <gln-option value="orange">Orange</gln-option>
  <gln-option value="kiwi">Kiwi</gln-option>
</gln-select>
