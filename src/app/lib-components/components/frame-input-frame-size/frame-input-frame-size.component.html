Frame Size

The "frameSize" property determines the size of the element (its height).

It takes the following values:

  • "short" - sets the height to 38px;
  • "small" - sets the height to 44px;
  • "middle" - sets the height to 50px (default);
  • "wide" - sets the height to 56px;
  • "large" - sets the height to 62px;
  • "huge" - sets the height to 68px;

The element's height is set according to the value of the frameSize parameter. All padding and offsets are determined by this variable and the font size.

The font size for native input and label is determined from the parent container. The size of the font can be defined with the css-style for the "grn-input" tag. In the example, this is implemented with the styles "fifs-large" and "fifs-huge". We can change the font size at the grn-input tag level, but the height of the element will not change.

The height of the element remains the same even when the font size is changed (from font-size: 10px to font-size: 26px).

It is possible to specify the height of an element. To do this, in the "config.frameSizeValue" property, we define the required height in pixels. In this case, "frameSize" and "config.frameSize" should not be used.

In order to hide the label (legend), add the "hiddenLabel" attribute.
