Frame Size #

The "frameSize" property determines the size of the element (its height).

It takes the following values:

  • "short" - sets the height to 38px;
  • "small" - sets the height to 44px;
  • "middle" - sets the height to 50px (default);
  • "wide" - sets the height to 56px;
  • "large" - sets the height to 62px;
  • "huge" - sets the height to 68px;

The element's height is set according to the value of the frameSize parameter. All padding and offsets are determined by this variable and the font size.

The font size for native input and label is determined from the parent container. The size of the font can be defined with the css-style for the "gln-input" tag. In the example, this is implemented with the styles "fs-large" and "fs-huge". We can change the font size at the gln-input tag level, but the height of the element will not change.

The height of the element remains the same even when the font size is changed (from font-size: 10px to 26px).

It is possible to specify the height of an element. To do this, in the "config.frameSizeValue" property, we define the required height in pixels. In this case "parameter frameSize" and "config.frameSize" should not be used.

In order to hide the label (legend), add the "isNoLabel" attribute.

exterior: {{ labelOutlined }} {{ labelUnderline }} {{ labelStandard }}
"Short" height: 38px; font-size: 16px;
{{ labelShowSource }} 
<form [formGroup]="formGroup02"
  novalidate
  autocomplete="off">

  <div class="app-group">
    <div class="app-elem fs-indent">
      <gln-input formControlName="model02a"
        label="Enter value"
        [isNoLabel]="isNoLabel02a ? '':null"
        frameSize="short"
        [exterior]="exterior02a">
      </gln-input>
    </div>
  </div>

</form>
 
import {
  Component, ViewEncapsulation
} from '@angular/core';
import {
  FormControl, FormGroup
} from '@angular/forms';

@Component({
  selector: 'app-frame-input',
  templateUrl: './frame-input.component.html',
  styleUrls: ['./frame-input.component.scss'],
  encapsulation: ViewEncapsulation.None,
})
export class FrameComponent {
  public formGroup02a: FormGroup = new FormGroup({
    model02a: new FormControl('Demo Size Short', []),
  });
  public exterior02 = 'outlined';
  public isNoLabel02a = false;
}
 
.app-group {
  display: flex;
  flex-wrap: wrap;
}
.app-elem {
  min-width: 26ch;
  margin: 10px 10px;
}
.fs-indent {
  margin-top: 15px;
}
