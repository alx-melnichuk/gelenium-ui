Config. #

If your application will have a single style for GlnTextarea elements, then you can set a single configuration. To do this, you need to create an object with the GlnTextareaConfig interface and define the required values. Next, specify the resulting object in the list of providers for the entire application (at the level of the AppComponent). 

const glnTextareaConfigDefault: GlnTextareaConfig = {
  exterior: 'outlined',
  size: 'small',
};

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.scss'],
  providers: [
    { provide: GLN_TEXTAREA_CONFIG, useValue: glnTextareaConfigDefault }
  ],
})
export class AppComponent {
  title = 'demo';
}

After that, all "GlnTextarea" components will have the same configuration.

1. Configuration with "outlined" and "small". #

For example, let's prepare a configuration in which:

  • "exterior" will have the value "outlined";
  • "size" will have the value "small" (height to 44px);
const glnTextareaConfigDefault: GlnTextareaConfig = {
  exterior: 'outlined',
  size: 'small',
};

2. Configuration with "underline" and "wide". #

Suppose we have a provider in our application with "GlnTextareaConfig" and all textarea elements accept its default values. We can change these values for an individual element. To do this, you need to specify in this element those properties that do not match the provider's configuration.

For example, let's specify the following parameters: 

exterior="underline"
size="wide"
isRequired

In other words, there is the following order of determining the configuration parameters:

  • Standard values are taken. 
    exterior="outlined", // GlnFrameConfig
size="middle",       // GlnFrameConfig
isLabelShrink=false, // GlnFrameConfig
isNoAnimation=false, // GlnFrameConfig
isRequired=false,    // GlnFrameConfig

isError=false,
isReadOnly=false,
ornamLfAlign='default',
ornamRgAlign='default',
isPlaceholder=false,
  • Values from the default configuration provider (if there is one).
  • Values from textarea parameters (if there is one).

import {
  Component, ViewEncapsulation
} from '@angular/core';
import {
  FormControl, FormGroup
} from '@angular/forms';
import {
  GlnTextareaConfig, GLN_TEXTAREA_CONFIG
} from 'gelenium-ui';

const glnTextareaConfigDefault: GlnTextareaConfig = {
  exterior: 'outlined',
  size: 'small',
};

@Component({
  selector: 'app-textarea',
  templateUrl: './textarea.component.html',
  styleUrls: ['./textarea.component.scss'],
  encapsulation: ViewEncapsulation.None,
  providers: [{
    provide: GLN_TEXTAREA_CONFIG, useValue: glnTextareaConfigDefault
  }],
})
export class TextareaComponent {
  public formGroup07: FormGroup = new FormGroup({
    model07a: new FormControl('', []),
    model07b: new FormControl('Hello World', []),
  });
}
 
app-textarea {
  app-input {

      flex-wrap: wrap;
    }
    .app-elem {
      min-width: 26ch;
      margin: 10px 10px;
    }
  }
}
