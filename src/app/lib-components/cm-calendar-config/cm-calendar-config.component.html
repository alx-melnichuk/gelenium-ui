Config. #

If your application will have a single style for GlnCalendar elements, then you can set a single configuration. To do this, you need to create an object with the GlnCalendarConfig interface and define the required values. Next, specify the resulting object in the list of providers for the entire application (at the level of the AppComponent). 

import {
  GlnCalendarConfig, GLN_CALENDAR_CONFIG
} from 'gelenium-ui';

const glnCalendarConfigDefault: GlnCalendarConfig = {
  cellSize: 'wide',
  isHideOldDays: true,
};

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.scss'],
  providers: [
    { provide: GLN_CALENDAR_CONFIG, useValue: glnCalendarConfigDefault }
  ],
})
export class AppComponent {
  title = 'demo';
}

After that, all input elements based on the "GlnCalendar" component will have the same configuration.

1. Configuration with "cellSize: 'wide'", "isHideDayoff: true" and "isHideOldDays: true". #

For example, let's prepare a configuration in which:

  • "cellSize" will have the value 'wide';
  • "isHideDayoff" will have the value "true";
  • "isHideOldDays" will have the value "true";
const glnCalendarConfigDefault: GlnCalendarConfig = {
  cellSize: 'wide',
  isHideDayoff: true,
  isHideOldDays: true,
};

2. Parameter values: "cellSize=42", "isHideDayoff=false" and "isHideOldDays=false". #

Suppose we have a provider in our application with "GlnCalendarConfig" and all input elements accept its default values. We can change these values for an individual element. To do this, you need to specify in this element those properties that do not match the provider's configuration.

For example, let's specify the following parameters: 

  <gln-calendar
    [cellSize]="42"
    [isHideDayoff]=false
    [isHideOldDays]=false
    [value]="value08a">
  </gln-calendar>

In other words, there is the following order of determining the configuration parameters:

  • Standard values are taken (if there is one).
  • Values from the default configuration provider (if there is one).
  • Values from input parameters (if there is one).
1. Config: cellSize='wide', isHideDayoff=true,
isHideOldDays=true.

import {
  Component, ViewEncapsulation
} from '@angular/core';

import {
  GlnCalendarConfig, GLN_CALENDAR_CONFIG
} from 'gelenium-ui';

const glnCalendarConfigDefault: GlnCalendarConfig = {
  cellSize: 'wide',
  isHideOldDays: true,
};

@Component({
  selector: 'app-calendar',
  templateUrl: './calendar.component.html',
  styleUrls: ['./calendar.component.scss'],
  encapsulation: ViewEncapsulation.None,
  providers: [{
    provide: GLN_CALENDAR_CONFIG, useValue: glnCalendarConfigDefault
  }],
})
export class CalendarComponent {
  public selectedDate: Date = this.getSelectedDate();

  public value08a: Date | null = new Date(this.selectedDate);
  public value08b: Date | null = new Date(this.selectedDate);

  private getSelectedDate(): Date {
    const d1: Date = new Date();
    const result: Date = new Date(d1.getFullYear(), d1.getMonth(), d1.getDate(), 0, 0, 0, 0);
    result.setDate(result.getDate() + (d1.getDate() === 1 ? 1 : -1));
    return result;
  }
}
