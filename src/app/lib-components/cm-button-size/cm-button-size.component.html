Button Size #

The "size" property determines the size of the element (its height).

It takes the following values:

  • "short" - sets the height to 38px;
  • "small" - sets the height to 44px (default);
  • "middle" - sets the height to 50px;
  • "wide" - sets the height to 56px;
  • "large" - sets the height to 62px;
  • "huge" - sets the height to 68px;

The font size is determined from the parent container. The size of the font can be defined with the css-style for the "gln-button" tag. In the example, this is implemented with the styles "bs-large" and "bs-huge". We can change the font size at the gln-input tag level, but the height of the element will not change.

The height of the element remains the same even when the font size is changed (from font-size: 10px to font-size: 26px).

It is possible to specify the height of an element. To do this, in the "config.frameSizeValue" property, we define the required height in pixels. In this case, "frameSize" and "config.frameSize" should not be used.

  public getLabel(exterior: string): string {
    return exterior ?
      exterior.substring(0, 1).toUpperCase() + exterior.substring(1) : '';
  }
}
