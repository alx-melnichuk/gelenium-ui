Config #

If your application will have a single style for GlnSelect elements, then you can set a single configuration. To do this, you need to create an object with the GlnSelectConfig interface and define the required values. Next, specify the resulting object in the list of providers for the entire application (at the level of the AppComponent). 

const glnSelectConfigDefault: GlnSelectConfig = {
  exterior: 'underline',
  size: 'wide',
};

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.scss'],
  providers: [
    { provide: GLN_SELECT_CONFIG, useValue: glnSelectConfigDefault }
  ],
})
export class AppComponent {
  title = 'demo';
}

After that, all "GlnSelect" components will have the same configuration.

1. Configuration with "outlined" and "small". #

For example, let's prepare a configuration in which:

  • "exterior" will have the value "outlined";
  • "size" will have the value "small" (height to 44px);
const glnSelectConfigDefault: GlnSelectConfig = {
  exterior: 'outlined',
  size: 'small',
};

2. Configuration with "underline" and "wide". #

Suppose we have a provider in our application with "GlnSelectConfig" and all select elements accept its default values. We can change these values for an individual element. To do this, you need to specify in this element those properties that do not match the provider's configuration.

For example, let's specify the following parameters: 

size="wide"
exterior="underline"
isRequired
backdropClass='sc-backdrop'
isMultiple=true
overlayClasses=['sc-overlay-panel']
classes=['sc-panel']
visibleSize=6

In other words, there is the following order of determining the configuration parameters:

  • Standard values are taken. 
    exterior="outlined", // GlnFrameConfig
size="middle",  // GlnFrameConfig
isLabelShrink=false, // GlnFrameConfig
isNoAnimation=false, // GlnFrameConfig
isRequired=false,    // GlnFrameConfig

isCheckmark=false,
isError=false,
isMultiple=false,
isNoIcon=false,
isNoRipple=false,
isReadOnly=false,
ornamLfAlign='default',
ornamRgAlign='default',
overlayClasses=undefined;
classes=undefined;
position='start';
visibleSize=0
  • Values from the default configuration provider (if there is one).
  • Values from input parameters (if there is one).

import {
  Component, ViewEncapsulation
} from '@angular/core';
import {
  FormControl, FormGroup
} from '@angular/forms';
import {
  GlnSelectConfig, GLN_SELECT_CONFIG
} from 'gelenium-ui';

const glnSelectConfigDefault: GlnSelectConfig = {
  exterior: 'outlined',
  size: 'wide',
  backdropClass: 'sc-backdrop',
  isMultiple: true,
  overlayClasses: ['sc-overlay-panel'],
  classes: ['sc-panel'],
  visibleSize: 6,
};

@Component({
  selector: 'app-select',
  templateUrl: './select.component.html',
  styleUrls: ['./select.component.scss'],
  encapsulation: ViewEncapsulation.None,
  providers: [{ provide: GLN_SELECT_CONFIG, useValue: glnSelectConfigDefault }],
})
export class SelectComponent {
  public salad = [
    'lettuce (lettuce)',
    'leaf parsley',
    'mustard leaf',
    'swans',
    'sorrel',
    'spinach',
    'purslane',
    'chard (leaf beet)',
    'vegetable fennel',
    'arugula',
    'cucumber grass',
    'nettles',
  ];
  public control08a = {
    model08a: new FormControl([this.salad[1]], []),
    model08b: new FormControl([this.salad[1]], []),
  };
  public formGroup08a: FormGroup = new FormGroup(this.control08a);

  public capitalizeFirstLetter(value: string): string {
    return value ? value[0].toUpperCase() + value.slice(1) : '';
  }
}
