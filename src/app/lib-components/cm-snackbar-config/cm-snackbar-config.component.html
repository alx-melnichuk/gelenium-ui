Config #

If your application will have a single style for displaying notification elements, then you can set a single configuration. To do this, you need to create an object with the GlnSnackbarConfig interface and define the required values.
Next, in the root class of the application (AppComponent), during initialization, we set the required configuration for snackbars. 

import { GlnSnackbarService } from 'gelenium-ui';

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.scss'],
})
export class AppComponent implements OnInit {

  constructor(private snackbarService: GlnSnackbarService) {}

  public ngOnInit(): void {
    this.snackbarService.setConfig({
      horizontal: 'left',
      vertical: 'top',
    });
  }
}

After that, all notifications will have the same display style.

1. Configuration with horizontal="left", vertical="top". #

For example, let's prepare a configuration in which:

  • "horizontal" will have the value "left";
  • "overlayClasses" will have the value "panel-custom";
  • "vertical" will have the value "top";
  • "transition" will have the value "turn";
  • "wrapperClass" will have the value "wrapper-custom";
const glnSnackbarConfigDefault: GlnSnackbarConfig = {
  horizontal: 'left',
  overlayClasses: 'panel-custom',
  vertical: 'top',
  transition: 'turn',
  wrapperClass: 'wrapper-custom',
};

2. Configuration with horizontal="right", vertical="bottom". #

Suppose we have a provider in our application with "GlnSnackbarConfig" and all notifications accept its default values. We can change these values for an individual element. To do this, you need to specify in this element those properties that do not match the provider's configuration.

For example, let's specify the following parameters: 

horizontal="right"
vertical="bottom"
transition="blur"

In other words, there is the following order of determining the configuration parameters:

  • Standard values are taken. 
    duration=6000,
horizontal="center",
maxCount=6,
overlayClasses=undefined,
vertical="bottom",
viewContainerRef=undefined,
transition="grow",
wrapperClass=undefined
  • Values from the default configuration provider (if there is one).
  • Values from input parameters (if there is one).
1. Default config: horizontal="left", vertical="top".

import {
  Component, ViewEncapsulation
} from '@angular/core';

import { GlnSnackbarService } from 'gelenium-ui';

@Component({
  selector: 'app-snackbar',
  templateUrl: './snackbar.component.html',
  styleUrls: ['./snackbar.component.scss'],
  encapsulation: ViewEncapsulation.None,
})
export class SnackbarComponent {

  constructor(private snackbarService: GlnSnackbarService) {}

  public clickConfig(message: string, config: any): void {
    this.snackbarService.open(message, '', config);
  }
}
