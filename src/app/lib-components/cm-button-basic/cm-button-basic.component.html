Basic. #

To use this component, in your module you need to add: 
import { GlnButtonModule } from 'gelenium-ui';

The GlnButton component is a styled button.
It comes in three versions:

  • as a container (exterior="contained")
  • as an outline (exterior="outlined")
  • as text (exterior="text" - default)

The content of the glnButton component is determined by the content between its opening and closing tags. This can be a fragment of html elements that will be displayed as the content of the button.
Another option, the content of the hyperlink - in this case, the hyperlink will remain. In practice, there are often situations when a button is displayed on the design, but in terms of functionality it is a hyperlink. The second option is designed for such situations.
Buttons with the "contained" exterior are for important primary actions.
Buttons with the "outlined" exterior are for important secondary actions.
Buttons with the "text" exterior are for less important actions.
