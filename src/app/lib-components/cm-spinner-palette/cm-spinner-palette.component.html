Palette. #

The GlnSpinner component has the ability to change its color.

For example: 
<gln-spinner [glnColor]="{ primary: 'danger' }">
</gln-spinner>

This means that you want to use the system color "danger" as the "primary" color.

To use the glnColor directive, you need to add in your module: 
import { GlnColorModule } from 'gelenium-ui';
color: primary
color: danger
color: success
color: info
color: warning
color: customer
