Basic. #

To use this component, in your module you need to add: 
import { GlnSpinnerModule } from 'gelenium-ui';

The GlnSpinner component displays a "svg" image that rotates around its own axis.
This component is used to display the data processing process.

Item size. #

The "size" property determines the size of the element (its height).

It takes the following values:

  • "short" - sets the height to 38px;
  • "small" - sets the height to 44px;
  • "middle" - sets the height to 50px (default);
  • "wide" - sets the height to 56px;
  • "large" - sets the height to 62px;
  • "huge" - sets the height to 68px;
"Short" height: 38px;
"Small" height: 44px;
"Middle" height: 50px;

"Large" height: 62px;

custom height 57px;

Palette. #

The GlnSpinner component has the ability to change its color.

For example: 
<gln-spinner
  [glnColor]="{ primary: 'danger' }">
</gln-spinner>

This means that you want to use the system color "danger" as the "primary" color.

To use the glnColor directive, you need to add in your module: 
import { GlnColorModule } from 'gelenium-ui';
color: primary
