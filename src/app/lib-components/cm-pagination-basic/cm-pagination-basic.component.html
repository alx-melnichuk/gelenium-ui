Basic. #

To use this component, in your module you need to add: 
import { GlnPaginationModule } from 'gelenium-ui';

The GlnPagination component is designed to change the current page from the specified range.

For example: 
<gln-pagination
  [page]="page01"
  [count]="pageCount01"
  (changePage)="page01a=$event">
</gln-pagination>
 The "page" current page number.
The "count" - total number of pages.
The "changePage" parameter is the page number change event.
Exterior: 'text'.
Exterior: 'outlined'.

Attributes. #

The following attributes are supported:

  • "id" - parameter for defining the unique identifier of the input field (id="unique_identifier");
  • "config" - Defines a configuration object. More details.
  • "count" - total number of pages; (default 1)
  • "page" - current page number; (default 1)
  • "exterior" - Determines the appearance of an element.
    It can take the following values:
    • "text" - default;
    • "outlined";
  • "isDisabled" - a sign that the field is disabled (isDisabled);
  • "isHidePrev" - indicates that the "previous page" button is hidden (isHidePrev);
  • "isHideNext" - indicates that the "next page" button is hidden (isHideNext);
  • "isShowFirst" - indicates that the "first page" button is displayed (isShowFirst);
  • "isShowLast" - indicates that the "last page" button is displayed (isShowLast);

Size #

The "size" property determines the size of the element (its height).

It takes the following values:

  • "short" - sets the height to 38px (default);
  • "small" - sets the height to 44px;
  • "middle" - sets the height to 50px;
  • "wide" - sets the height to 56px;
  • "large" - sets the height to 62px;
  • "huge" - sets the height to 68px;

The font size is determined from the parent container. The size of the font can be defined with the css-style for the "gln-pagination" tag. We can change the font size at the gln-pagination tag level, but the height of the element will not change.

The height of the element remains the same even when the font size is changed (from font-size: 10px to font-size: 26px).

It is possible to specify the height of an element. To do this, in the "config.size" property, we define the required height in pixels.

Border #

The GlnPagination component has the ability to change the border-radius css property.
In other words, the buttons of the GlnPagination component can be square instead of round.
The round shape of the GlnPagination component buttons is given a special css property '--glnbt-br-rd'.
If you exclude this css-property, then the shape of the buttons will be square.
To disable this css property, you need to add the 'isNoRound' attribute.

Attributes: isNoRound, Exterior: 'text'.
Config: isNoRound, Exterior: 'outlined'.
{{ labelShowSource }} 
<div class="app-group">

  <div class="app-elem pgr-elem">
    <div>
      <small>
        <em>Attributes: isNoRound, Exterior: 'text'.</em>
      </small>
    </div>
    <gln-pagination [page]="page04a"
      style="font-size: 16px;"
      [count]="10"
      exterior="text"
      isNoRound
      (changePage)="page04a = $event">
    </gln-pagination>
  </div>

  <div class="app-elem pgr-elem">
    <div>
      <small>
        <em>Config: isNoRound, Exterior: 'outlined'.</em>
      </small>
    </div>
    <gln-pagination [page]="page04b"
      style="font-size: 16px;"
      [count]="10"
