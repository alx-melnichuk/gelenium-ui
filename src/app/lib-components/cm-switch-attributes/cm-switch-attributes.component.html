Attributes. #

The following attributes are supported:

  • "id" - parameter for defining the unique identifier of the input field (id="unique_identifier");
  • "config" - Defines a configuration object. More details.
  • "isChecked" - specifies only the initial value of the element (isChecked);
  • "isDisabled" - a sign that the field is disabled (isDisabled);
  • "isNoAnimation" - disable animation execution (isNoAnimation);
  • "isNoRipple" - disables the execution of the ripple effect (isNoRipple);
  • "isReadOnly" - Specifies the status of an element as read-only (isReadOnly);
