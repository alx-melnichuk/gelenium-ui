Basic. #

To use this component, in your module you need to add: 
import { GlnDatepickerModule } from 'gelenium-ui';

The GlnDatepicker component is a form control for picking a date.
It includes: a label, a list of options, and a help text.

It comes in three versions:
  • with an outline (exterior="outlined" - default)
  • with a fill (exterior="underline")
  • standard (exterior="standard")
For example: 
<gln-datepicker formControlName="model01a"
  label="Enter value">
</gln-datepicker>

Size #

The "cell-size" property determines the cell size of the element (its height and width).

It takes the following values:

  • "short" - sets the height to 28px;
  • "small" - sets the height to 32px;
  • "middle" - sets the height to 36px (default);
  • "wide" - sets the height to 40px;
  • "large" - sets the height to 44px;
  • "huge" - sets the height to 48px;
"Short" height: 28px; font-size: 14px;
"Small" height: 32px; font-size: 15px;
"Middle" height: 36px; font-size: 16px;
"Wide" height: 40px; font-size: 16px;
"Large" height: 44px; font-size: 18px;
"Huge" height: 48px; font-size: 20px;
