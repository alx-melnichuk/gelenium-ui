Basic. #

To use this component, in your module you need to add: 
import { GlnRadioButtonModule, GlnRadioGroupModule } from 'gelenium-ui';

GlnRadioButton components are typically used in groups that represent possible values for a single parameter. Components are grouped by a unique "name" attribute value that is the same for all components in this group. In this case, only one value can be selected.

The GlnRadioButton component can be used in reactive forms.

For example: 
<form [formGroup]="formGroup01a" novalidate autocomplete="off">
  <gln-radio-button *ngFor="let item of fruits01a"
    formControlName="model01a"
    name="model01a"
    [value]="item"> {{ item }}
  </gln-radio-button>
</form>
With "formControlName" model="{{ formGroup01a.controls['model01a'].value }}"
{{ item }}

import {
  Component, ViewEncapsulation
} from '@angular/core';
import {
  FormControl, FormGroup
} from '@angular/forms';

@Component({
  selector: 'app-radio-button',
  templateUrl: './radio-button.component.html',
  styleUrls: ['./radio-button.component.scss'],
  encapsulation: ViewEncapsulation.None,
})
export class RadioButtonComponent {
  public fruits01a: string[] = ['apple', 'apricot', 'banana'];
  public formGroup01a: FormGroup = new FormGroup({
    model01a: new FormControl(this.fruits01a[0], []),
  });
}

And also the GlnRadioButton component can be used without reactive forms.

For example: 
<gln-radio-button *ngFor="let item of fruits01b"
  [(ngModel)]="data01b"
  name="model01b"
  [value]="item"> {{ item }}
</gln-radio-button>
With "ngModel" model="{{ data01b }}"
{{ item }}
{{ labelShowSource }} 
<div class="app-group">

  <div class="app-elem">
    <div>
      <small>
        <em>With "ngModel" model="{{ data01b }}"</em>
      </small>
    </div>
    <gln-radio-button *ngFor="let item of fruits01b"
      name="model01b"
      [(ngModel)]="data01b"
      [value]="item">
      {{ item }}
    </gln-radio-button>

  </div>

</div>

The GlnRadioGroup component allows you to select one value from a group of GlnRadioButton elements. And also apply settings attributes immediately for the entire group of GlnRadioButton elements.

For example: 
<form [formGroup]="formGroup01c" novalidate autocomplete="off">
  <gln-radio-group name="model01c" label="Choose a fruit:">
    <gln-radio-button *ngFor="let item of fruits01c"
      formControlName="model01c"
      [value]="item"> {{ item }}
    </gln-radio-button>
  </gln-radio-group>
</form>
With GlnRadioGroup model="{{ formGroup01c.controls['model01c'].value }}"
{{ item }}
