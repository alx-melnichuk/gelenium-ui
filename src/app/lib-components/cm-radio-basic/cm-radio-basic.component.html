Basic. #

To use this component, in your module you need to add: 
import { GlnRadioButtonModule, GlnRadioGroupModule } from 'gelenium-ui';

The GlnRadioButton component displays the behavior of a radio button in a group of related options (<input type="radio">). The component allows you to select only one element from several related components. The GlnRadioButton component is used inside the GlnRadioGroup component.

The GlnRadioButton component can be used in reactive forms.

For example: 
<form [formGroup]="formGroup01a" novalidate autocomplete="off">
  <gln-radio-button *ngFor="let item of fruits01a"
    formControlName="model01a"
    name="model01a"
    [value]="item"> {{ item }}
  </gln-radio-button>
</form>
With "formControlName" model="{{ formGroup01a.controls['model01a'].value }}""
{{ item }}

And also the GlnRadioButton component can be used without reactive forms.

For example: 
<gln-radio-button *ngFor="let item of fruits01b"
  [(ngModel)]="data01b"
  name="model01b"
  [value]="item"> {{ item }}
</gln-radio-button>
With "ngModel" model="{{ data01b }}"
{{ item }}

The GlnRadioGroup component allows you to select one value from a group of GlnRadioButton elements. And also apply settings attributes immediately for the entire group of GlnRadioButton elements.

For example: 
<form [formGroup]="formGroup01c" novalidate autocomplete="off">
  <gln-radio-group>
    <gln-radio-button *ngFor="let item of fruits01c"
      formControlName="model01c"
      name="model01c"
      [value]="item"> {{ item }}
    </gln-radio-button>
  </gln-radio-group>
</form>
With GlnRadioGroup model="{{ formGroup01c.controls['model01c'].value }}"

Attributes. #

The following attributes are supported:

  • "id" - parameter for defining the unique identifier of the input field (id="unique_identifier");
  • "config" - Defines a configuration object. More details.
  • "isDisabled" - a sign that the field is disabled (isDisabled);
  • "isReadOnly" - Specifies the status of an element as read-only (isReadOnly);
  • "isNoRipple" - disables the execution of the ripple effect (isNoRipple);
  • "isNoHover" - disables the display of additional background on hover (isNoHover);

Size. #

The GlnRadioButton component has the ability to change its size. The size of the image is related to the font size of the inner element with the class "gln-radio-icon-block". To change the size of the component image, you need to specify the "size" parameter. Or specify the required font size for the block with the class "gln-radio-icon-block" (you can specify the css parameter "--glnrb-icon-fn-sz").
