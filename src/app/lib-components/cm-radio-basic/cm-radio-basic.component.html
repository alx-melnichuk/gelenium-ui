Basic. #

To use this component, in your module you need to add: 
import { GlnRadioButtonModule, GlnRadioGroupModule } from 'gelenium-ui';

GlnRadioButton components are typically used in groups that represent possible values for a single parameter. Components are grouped by a unique "name" attribute value that is the same for all components in this group. In this case, only one value can be selected.

The GlnRadioButton component can be used in reactive forms.

For example: 
<form [formGroup]="formGroup01a" novalidate autocomplete="off">
  <gln-radio-button *ngFor="let item of fruits01a"
    formControlName="model01a"
    name="model01a"
    [value]="item"> {{ item }}
  </gln-radio-button>
</form>
With "formControlName" model="{{ formGroup01a.controls['model01a'].value }}""

Attributes. #

The following attributes are supported:

  • "id" - parameter for defining the unique identifier of the input field (id="unique_identifier");
  • "config" - Defines a configuration object. More details.
  • "isDisabled" - a sign that the field is disabled (isDisabled);
  • "isReadOnly" - Specifies the status of an element as read-only (isReadOnly);
  • "checked" - indicates that the current element is selected by default (used if ngModel and Reactive Forms are not used) (isNoHover);

Size. #

The GlnRadioButton component has the ability to change its size. The size of the image is related to the font size of the inner element with the class "gln-radio-icon-block". To change the size of the component image, you need to specify the "size" parameter. Or specify the required font size for the block with the class "gln-radio-icon-block" (you can specify the css parameter "--glnrb-icon-fn-sz").

The GlnRadioButton component has the ability to change its color.

For example: 
<gln-radio-button formControlName="model07" name="model07"
  [glnColor]="{ primary: 'danger' }"
  value="demo1">Demo 1
</gln-radio-button>

This means that you want to use the system color "danger" as the "primary" color.

To use the glnColor directive, you need to add in your module: 
import { GlnColorModule } from 'gelenium-ui';
