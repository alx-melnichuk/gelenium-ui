Basic. #

To use this component, in your module you need to add: 
import { GlnRadioModule } from 'gelenium-ui';

The GlnRadio component displays the behavior of a radio button in a group of related options (<input type="radio">). The component allows you to select only one element from several related components.

For example: 
<gln-radio formControlName="model01">
</gln-radio>
