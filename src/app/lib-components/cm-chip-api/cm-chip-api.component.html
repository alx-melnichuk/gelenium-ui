  | undefined; 		This property determines the size of the element (its height).
It takes the following values:
  • "short" - sets the height to 24px;
  • "little" - sets the height to 28px;
  • "small" - sets the height to 32px (default);
  • "middle" - sets the height to 36px;
  • "wide" - sets the height to 40px;
This property can contain the element's height in pixels, either as a string or as a number.
For example: size="30" or [size]="30"

CSS variables at the component level. #

Name Description
--glnch--default-[h,s,l]:
var(--glncl-default-
[h,s,l]); 		This setting defines:
  • text color in ripple effect;
  • frame color (for "outlined" exterior);
  • background color when hovering the mouse;
The value is determined by the system css-variable --gln-default-[h,s,l].
Or the css-variable of the GlnColor directive --glncl-default-[h,s,l].
Parameters [h,s,l] defines the value of "hue", "saturation", "lightness".

Css variables, optional. You can override the default values. #
