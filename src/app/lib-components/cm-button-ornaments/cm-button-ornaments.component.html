Ornaments. #

For the GlnButton component, it is possible to display an ornament (picture or text).
To display the ornament on the left, you need to add a block with the "gln-orn-lf" attribute inside the "gln-button".
To display the ornament on the right, you need to add a block with the "gln-orn-rg" attribute inside the "gln-button".

Example: 
<gln-button>
  <div gln-orn-lf><span>left</span></div>
  text
  <div gln-orn-rg><span>right</span></div>
</gln-button>

For blocks: <span>left</span> and <span>right</span> you need to specify the value of horizontal indents.

Button contains a link.

When a GlnButton is in "link" mode, its content is only determined by the content of the "a" html tag.
This means that the ornament can be rendered as an inner element of the html "a" tag.
For example, inside the "a" tag, there are two consecutive span elements. The first span element contains the name of the element ("Next link"). The second span element contains an ornament - text or picture.

Example, ornament on the right: 
<gln-button>
  <a href="#">
    <span>Next link</span>
    <span class="bo-text">  &gt;  </span>
  </a>
  <i gln-orn-rg></i>
</gln-button>
 Example, ornament on the left: 
<gln-button>
  <a href="#">
    <span class="bo-text">  &lt;  </span>
    <span>Previous link</span>
  </a>
  <i gln-orn-lf></i>
</gln-button>

Ornament - text. #

The css class "bo-text" defines the horizontal padding. The css class "bo-font-large" defines the font size as 20px.
The "<i gln-orn-rg></i>" snippet removes padding from the right of the GlnButton.
