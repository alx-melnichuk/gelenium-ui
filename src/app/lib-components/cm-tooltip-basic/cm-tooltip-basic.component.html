Basic. #

To use this component, in your module you need to add: 
import { GlnTooltipModule } from 'gelenium-ui';
 Tooltip displays text when the user:
  • hover over an element;
  • focuses on the element;
  • by clicking on an element;
For example: 
<button glnTooltip="Information about the action.">
  Action
</button>
Show tooltip on hover.
Show tooltip with focus.
Show tooltip on click.

Attributes. #

The following attributes are supported:
  • "config" - Defines a configuration object. More details.
  • "glnTooltip" - specifies the tooltip text;
  • "glnTooltipPosition" - specifies the position of the tooltip;
    This property can take the following values:
    • 'bottom' (default);
    • 'bottom-start';
    • 'bottom-end';
    • 'top';
    • 'top-start';
    • 'top-end';
    • 'left';
    • 'left-start';
    • 'left-end';
    • 'right';
    • 'right-start';
    • 'right-end';

Attributes are also supported 2: #

  • "glnTooltipShowDelay" - specifies the delay time before showing the tooltip, ms;
  • "glnTooltipHideDelay" - specifies the delay time before hiding the tooltip, ms;
