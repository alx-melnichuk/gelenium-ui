Attributes. #

The following attributes are supported:

  • "id" - parameter for defining the unique identifier of the input field (id="unique_identifier");
  • "config" - Defines a configuration object. More details.
  • "count" - total number of pages; (default 1)
  • "page" - current page number; (default 1)
  • "exterior" - Determines the appearance of an element.
    It can take the following values:
    • "text" - default;
    • "outlined";
  • "isDisabled" - a sign that the field is disabled (isDisabled);
  • "isHidePrev" - indicates that the "previous page" button is hidden (isHidePrev);
  • "isHideNext" - indicates that the "next page" button is hidden (isHideNext);
  • "isShowFirst" - indicates that the "first page" button is displayed (isShowFirst);
  • "isShowLast" - indicates that the "last page" button is displayed (isShowLast);

Attributes are also supported 2: #

  • "countBorder" - number of always visible pages at the beginning and at the end;
    (default 1)
  • "countNearby" - number of always visible pages before and after the current page;
    (default 1)
  • "isNoRipple" - This property to turn off the ripple effect;
    (default false)
