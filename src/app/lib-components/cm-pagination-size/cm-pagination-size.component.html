Size #

The "size" property determines the size of the element (its height).

It takes the following values:

  • "short" - sets the height to 38px (default);
  • "small" - sets the height to 44px;
  • "middle" - sets the height to 50px;
  • "wide" - sets the height to 56px;
  • "large" - sets the height to 62px;
  • "huge" - sets the height to 68px;

The font size is determined from the parent container. The size of the font can be defined with the css-style for the "gln-pagination" tag. We can change the font size at the gln-pagination tag level, but the height of the element will not change.

The height of the element remains the same even when the font size is changed (from font-size: 10px to font-size: 26px).

It is possible to specify the height of an element. To do this, in the "config.size" property, we define the required height in pixels.
