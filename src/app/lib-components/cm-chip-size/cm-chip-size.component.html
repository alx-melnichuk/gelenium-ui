Size #

The "size" property determines the size of the element (its height).

It takes the following values:
  • "short" - sets the height to 24px;
  • "little" - sets the height to 28px;
  • "small" - sets the height to 32px (default);
  • "middle" - sets the height to 36px;
  • "wide" - sets the height to 40px;

The height of the "delete" button is determined by the formula: Math.round(size * 0.672).

It is possible to specify the height of an element. To do this, in the "config.size" property, we define the required height in pixels.
