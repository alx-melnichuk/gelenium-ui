Size. #

The "size" property determines the size of the element (its height).

It takes the following values:

  • "short" - sets the height to 38px;
  • "small" - sets the height to 44px (default);
  • "middle" - sets the height to 50px;
  • "wide" - sets the height to 56px;
  • "large" - sets the height to 62px;
  • "huge" - sets the height to 68px;
"Short" height: 38px;

Configuration for custom height (57px). #

In the 'size' parameter, you can specify the required size of the element.

For example: 
<gln-spinner size="57">
</gln-spinner>

You can also specify the required size of the element in the configuration parameter.

For example: 
<gln-spinner [config]="{ size: 57 }">
</gln-spinner>
custom [size]="57"
custom [config]="{ size: '57' }"
