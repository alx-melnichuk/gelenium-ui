Basic. #

To use this component, in your module you need to add: 
import { GlnCalendarModule } from 'gelenium-ui';

This component displays the calendar month (a grid of days of the month) and allows you to select a date. With ngModel, the specified value comes after the second call to writeValue(), which happens after AfterViewInit().
ValueAccessor.writeValue is being called twice, first time with a phantom null value Therefore, the component does not support the ControlValueAccessor interface.

For example: 
<gln-calendar
</gln-calendar>
Model: {{ value01a == null ? 'null' : value01a | date:'YYYY/MM/dd' }}
