Basic. #

To use this component, in your module you need to add: 
import { GlnCalendarModule } from 'gelenium-ui';

This component displays the calendar month (a grid of days of the month) and allows you to select a date.

Here is an example of usage: 
<gln-calendar [value]="model01">
</gln-calendar>
 
public model01: Date | null = new Date();
Attribute: value
Model: {{ value01a == null ? 'null' : value01a | date:'YYYY/MM/dd' }}
source

The component does not support the ControlValueAccessor interface.
This is due to the fact that for rendering it is required to get the selected date, and if ngModel is used, then the selected date comes only with a writeValue() call and this happens after AfterViewInit().
ValueAccessor.writeValue is being called twice, first time with a phantom null value
In other words, the data for rendering arrives too late.
