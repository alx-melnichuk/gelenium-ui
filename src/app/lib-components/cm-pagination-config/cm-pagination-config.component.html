Config. #

If your application will have a single style for all GlnPagination elements, then you can set a single configuration. To do this, you need to create an object with the GlnPaginationConfig interface and define the required values. Next, specify the resulting object in the list of providers for the entire application (at the level of the AppComponent). 

const glnPaginationConfigDefault: GlnPaginationConfig = {
  countNearby: 2,
  exterior: 'outlined',
  size: 'small',
};

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.scss'],
  encapsulation: ViewEncapsulation.None,
  providers: [
    { provide: GLN_PAGINATION_CONFIG, useValue: glnPaginationConfigDefault }
  ],
})
export class AppComponent {
  title = 'demo';
}

After that, all GlnPagination elements based on the "GlnPaginationConfig" component will have the same configuration.

1. Configuration with "outlined" and "small". #

For example, let's prepare a configuration in which:

  • "countNearby" will have the value 2.
  • "exterior" will have the value "outlined".
  • "size" will have the value "small" (height to 44px;).
const glnPaginationConfigDefault: GlnPaginationConfig = {
  count: 11,
  countNearby: 2,
  exterior: 'outlined',
  size: 'small',
};

2. Specify "text" and "middle". #

Suppose we have a provider in our application with "GlnPaginationConfig" and all GlnPagination elements accept its default values. We can change these values for an individual element. To do this, you need to specify in this element those properties that do not match in the configuration provider.

For example, let's specify the following parameters: 

  count: 12,
  countNearby: 1,
  exterior="text",
  size="middle"

In other words, there is the following order of determining the configuration parameters:

  • Standard values are taken. 
      count=1,
  countBorder=1,
  countNearby=1,
  exterior="text",
  isHideNext=false,
  isHidePrev=false,
  isNoRounded=false,
  isShowFirst=false,
  isShowLast=false,
  size="short"
  • Values from the default configuration provider (if there is one).
  • Values from input parameters (if there is one).
1. Default config: "outlined", "small", countNearby=2, count=11.
2. exterior="text", size="middle", countNearby=1, count=12.

import {
  Component, ViewEncapsulation
} from '@angular/core';
import {
  GlnPaginationConfig, GLN_PAGINATION_CONFIG
} from 'gelenium-ui';

const glnPaginationConfigDefault: GlnPaginationConfig = {
  count: 11,
  countNearby: 2,
  exterior: 'outlined',
  size: 'small',
};

@Component({
  selector: 'app-pagination',
  templateUrl: './pagination.component.html',
  styleUrls: ['./pagination.component.scss'],
  encapsulation: ViewEncapsulation.None,
  providers: [
    { provide: GLN_PAGINATION_CONFIG, useValue: glnPaginationConfigDefault }
  ],
})
export class PaginationComponent {
  public page08a: number = 6;
  public page08b: number = 6;
}
