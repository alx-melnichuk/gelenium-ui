Config. #

If your application will have a single style for GlnSwitch elements, then you can set a single configuration. To do this, you need to create an object with the GlnSwitchConfig interface and define the required values. Next, specify the resulting object in the list of providers for the entire application (at the level of the AppComponent). 

import {
  GlnSwitchConfig, GLN_SWITCH_CONFIG
} from 'gelenium-ui';

const glnSwitchConfigDefault: GlnSwitchConfig = {
  isNoRipple: true,
  position: 'start',
};

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.scss'],
  providers: [
    { provide: GLN_SWITCH_CONFIG, useValue: glnSwitchConfigDefault }
  ],
})
export class AppComponent {
  title = 'demo';
}

After that, all input elements based on the "GlnSwitch" component will have the same configuration.

1. Configuration with "isNoRipple: true" and "position: start". #

For example, let's prepare a configuration in which:

  • "isNoRipple" will have the value "true";
  • "position" will have the value "start";).
const glnSwitchConfigDefault: GlnSwitchConfig = {
  isNoRipple: true,
  position: 'start',
};

2. Parameter values: "isNoRipple=false" and "position=end". #

Suppose we have a provider in our application with "GlnurlCmSwitchConfig" and all input elements accept its default values. We can change these values for an individual element. To do this, you need to specify in this element those properties that do not match the provider's configuration.

For example, let's specify the following parameters: 

  <gln-switch
    isNoRipple="false"
    position="end"
    formControlName="model06b">Label
  </gln-switch>

In other words, there is the following order of determining the configuration parameters:

  • Standard values are taken (if there is one).
  • Values from the default configuration provider (if there is one).
  • Values from input parameters (if there is one).

import {
  Component, ViewEncapsulation
} from '@angular/core';
import {
  FormControl, FormGroup
} from '@angular/forms';

import {
  GlnSwitchConfig, GLN_SWITCH_CONFIG
} from 'gelenium-ui';

const glnSwitchConfigDefault: GlnSwitchConfig = {
  isNoRipple: true,
  position: 'start',
};

@Component({
  selector: 'app-switch',
  templateUrl: './switch.component.html',
  styleUrls: ['./switch.component.scss'],
  encapsulation: ViewEncapsulation.None,
  providers: [{
    provide: GLN_SWITCH_CONFIG, useValue: glnSwitchConfigDefault
  }],
})
export class SwitchComponent {
  public formGroup06: FormGroup = new FormGroup({
    model06a: new FormControl(true, []),
    model06b: new FormControl(true, []),
  });
}
 
app-switch {
  .app-group {
    display: flex;
    flex-wrap: wrap;
  }
  .app-elem {
    min-width: 26ch;
    margin: 10px 10px;
  }
}
