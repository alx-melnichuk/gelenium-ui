Ornaments. #

For the GlnPagination component, it is possible to display an ornament (picture or text).
To display the ornament, you need to add a block inside "gln-pagination" with one of the attributes: "gln-orn-first", "gln-orn-prev", "gln-orn-next", "gln-orn-last".

Example: 
<gln-pagination>
  <span gln-orn-prev>prev</span>
  <span gln-orn-next>next</span>
  <span gln-orn-first>first</span>
  <span gln-orn-last>last</span>
</gln-pagination>

For these blocks, you need to specify a horizontal indent value or use the 'glnpg-pd-hor' attribute.
