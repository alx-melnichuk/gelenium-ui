Basic. #

To use this component, in your module you need to add: 
import { GlnSwitchModule } from 'gelenium-ui';

The GlnSwitch component displays the "on" or "off" state. It can be used to display the status of any parameter.

unchecked by default.

Attributes. #

The following attributes are supported:

  • "id" - parameter for defining the unique identifier of the input field (id="unique_identifier");
  • "config" - Defines a configuration object.
For example: 
<gln-switch formControlName="model02a">
  Label
</gln-switch>

Size. #

The GlnSwitch component has the ability to change its size. All component elements are associated with a font size for the gln-switch tag. Thus, by changing the font size, you can choose the size of this component that is optimal for you.

Color. #

The GlnSwitch component has the ability to change its color.

For example: 
<gln-switch
  [glnColor]="{ primary: 'danger' }"
  formControlName="model02a">
  Label
</gln-switch>
