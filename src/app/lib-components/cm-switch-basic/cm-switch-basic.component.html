Basic. #

To use this component, in your module you need to add: 
import { GlnSwitchModule } from 'gelenium-ui';

The GlnSwitch component displays an "on" or "off" state. This component is designed to display the state of any parameter.

checked by default.
unchecked by default.
disabled and checked.
disabled and unchecked.
