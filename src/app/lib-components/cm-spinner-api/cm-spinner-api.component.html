  | boolean
  | null

@Input() size: number
  | string
  | null
  | undefined; 		Determines the size of the element.
It can take the following values:
  • "short" - height: 38px;
  • "small" - height: 44px (default);
  • "middle" - height: 50px;
  • "wide" - height: 56px;
  • "large" - height: 62px;
  • "huge" - height: 68px;
This property can also contain a string with the value of the height of the element in pixels.
For example: size="57" or [size]="57"

Css variables, optional. You can override the default values. #
