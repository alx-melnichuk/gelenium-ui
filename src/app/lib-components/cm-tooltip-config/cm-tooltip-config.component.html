Config #

If your application will have a single style for GlnTooltip elements, then you can set a single configuration. To do this, you need to create an object with the GlnTooltipConfig interface and define the required values. Next, specify the resulting object in the list of providers for the entire application (at the level of the AppComponent). 

const glnTooltipConfigDefault: GlnTooltipConfig = {
  classes: 'ttc-demo',
  isArrow: true,
};

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.scss'],
  providers: [
    { provide: GLN_TOOLTIP_CONFIG, useValue: glnTooltipConfigDefault }
  ],
})
export class AppComponent {
  title = 'demo';
}

After that, all "GlnTooltip" components will have the same configuration.

1. Configuration with isArrow="false", Position="bottom-start", Classes="ttcf-panel". #

For example, let's prepare a configuration in which:

  • "classes" will have the value "ttcf-panel";
  • "isArrow" will have the value "true";
  • "isNoTransform" will have the value "true";
  • "position" will have the value "bottom-start";
const glnTooltipConfigDefault: GlnTooltipConfig = {
  classes: 'ttcf-panel',
  isArrow: true,
  isNoTransform: true,
  position: 'bottom-start',
};

2. Configuration with isArrow="false", Position="right", Classes="_", MaxWidth="120". #

Suppose we have a provider in our application with "GlnTooltipConfig" and all tooltip elements accept its default values. We can change these values for an individual element. To do this, you need to specify in this element those properties that do not match the provider's configuration.

For example, let's specify the following parameters: 

classes="_"
maxWidth="120"
isArrow="false"
position="right"
isNoTransform="false"

In other words, there is the following order of determining the configuration parameters:

  • Standard values are taken. 
    classes=undefined,
hideDelay=0,
hideTouchDelay=1500,
isArrow=false,
isNoAnimation=false,
isNoHideOnScroll=false,
isNoHoverable=false,

isNoTransform=false,
maxHeight=undefined,
maxWidth=280,
minHeight=undefined,
minWidth=undefined,
overlayClasses=undefined,
position='bottom',
showDelay=100,
showTouchDelay=800
  • Values from the default configuration provider (if there is one).
  • Values from input parameters (if there is one).

import {
  Component, ViewEncapsulation
} from '@angular/core';
import {
  GlnTooltipConfig, GLN_TOOLTIP_CONFIG
} from 'gelenium-ui';

const glnTooltipConfigDefault: GlnTooltipConfig = {
  classes: 'ttcf-panel',
  isArrow: true,
  isNoTransform: true,
  position: 'bottom-start',
  };

@Component({
  selector: 'app-tooltip',
  templateUrl: './tooltip.component.html',
  styleUrls: ['./tooltip.component.scss'],
  encapsulation: ViewEncapsulation.None,
  providers: [{ provide: GLN_TOOLTIP_CONFIG, useValue: glnTooltipConfigDefault }],
})
export class TooltipComponent {
}
