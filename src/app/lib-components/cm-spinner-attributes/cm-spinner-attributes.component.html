Attributes. #

The following attributes are supported:

  • "id" - parameter for defining the unique identifier of the input field (id="unique_identifier");
  • "config" - Defines a configuration object. More details.
  • "isExternal" - Displays the specified content instead of an inline SVG image.
  • "isNoAnimation" - disable animation execution (isNoAnimation);
  • "isNoPulsate" - disable circle pulsate (isNoPulsate);
  • "progress" - the value of the filled circle arc value from 0 to 1 (default 0.7);
  • "size" - specifies the size (height and width) of the component.
