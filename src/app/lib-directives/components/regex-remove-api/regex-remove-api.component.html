API: GlnRegexRemove

Selector: [glnRegexRemove]

Exported as: glnRegexRemove

When changing the value, the directive deletes the data that matches the specified regular expression.

Properties

Name Description
@Input() glnRegexRemove: string | null; Regular expression for entering numbers only.
  • glnRegexRemove="/[^\d]/gm"
  • [glnRegexRemove]="'/[^\\d]/gm'"
