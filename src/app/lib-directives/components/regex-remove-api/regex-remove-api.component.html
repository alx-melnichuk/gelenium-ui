API: GrnRegexRemove

Selector: grn-regex-remove

Exported as: grnRegexRemove

When changing the value, the directive deletes the data that matches the specified regular expression.

Properties

Name Description
@Input() grnRegexRemove: string | null; Regular expression for entering numbers only.
  • grnRegexRemove="/[^\d]/gm"
  • [grnRegexRemove]="'/[^\\d]/gm'"
