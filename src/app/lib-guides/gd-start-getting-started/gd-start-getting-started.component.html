Installation: Angular CLI way. #

Use the Angular CLI ng add command for updating your Angular project. 

$ ng add gelenium-ui

Installation: Manual way. #

Install Gelenium-UI from npm. 

$ npm install gelenium-ui --save

Description of the Gelenium-UI package at the gelenium-ui link.

Append: CSS. #

You need to add Gelenium-UI CSS to your application.

For example you could install Gelenium-UI from npm 

$ npm install gelenium-ui
and add Gelenium-UI CSS to your project.

You just need to add Gelenium-UI styles to your angular.json configuration: 

"yourApp": {
  "architect": {
    "build": {
      "options": {
        "styles": [
          "node_modules/gelenium-ui/styles.css"
        ]
      }
    }
  }
}

Import Gelenium-UI modules. #

Further, for all modules (whose components use elements of libraries), it is required to import the used library modules. 

import { GlnInputModule } from 'gelenium-ui';

@NgModule({
  imports: [ GlnInputModule ],
})
export class YourAppModule {
}
