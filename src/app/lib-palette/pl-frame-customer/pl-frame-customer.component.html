This library provides base colors for displaying components in various states: mouseover, focus, error, and disabled.

These colors are fixed in CSS variables:
  • "default-bg" default background color;
  • "default" default text color;
  • "primary" color when receiving focus;
  • "danger" color in error state;
  • "success" color in the state of success;
  • "info" color for displaying an information message;
  • "warning" color for displaying a warning message;

To display the GlnFrame component, use: default, primary, danger. Using the GlnColor directive, we can override the value of these CSS variables for this component.

To use the glnColor directive, you need to add in your module: 
import { GlnColorModule } from 'gelenium-ui';
