Basic

By appearance and behavior, a group of components can be distinguished: GrnInput, GrnTextarea.

They have the same functionality:

  • display of the frame of the appearance;
  • the size of the indents;
  • animation (shrink) of the label;
  • helper text;

The implementation of these functions has been moved into a separate auxiliary component "GrnFrameInput". This made it possible to: reduce development time, eliminate code duplication and exclude the human factor to miss something.

Let's consider the capabilities of the "GrnFrameInput" component using the "GrnInput" example. Some properties of "GrnInput" are passed directly to its internal component "GrnFrameInput".

The "exterior" property determines the appearance of the component.

It takes the following values:

  • "outlined" - a border in the form of an rectangular frame and horizontal indents;
  • "underline" - a border in the form of an underline frame and horizontal indents;
  • "standard" - a border in the form of an underline and without horizontal indents;
