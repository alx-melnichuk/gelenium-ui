Attributes

The following attributes are supported:

  • "id" - parameter for defining the unique identifier of the field;
  • "isReadOnly" - a sign that the field is read-only;
  • "isRequired" - a sign that the field is required;
  • "isDisabled" - a sign that the field is disabled;
  • "lbShrink" - a sign that the field has a permanent label, without being converted to a placeholder;
  • "hiddenLabel" - a sign that the field has no label at all;
  • "helperText" - the parameter is used to provide context for a given field or error message;
  • "isError" - a sign that the field is in error;
  • "minLength" - sets the minimum field length;
  • "maxLength" - sets the maximum field length;
  • "cntRows" - sets the number of lines that, when multiplied by the line height, determine the total height. If not specified, the height is determined by the number of lines in the text.
  • "minRows" - sets the number of lines that, when multiplied by the line height, determine the min height. Active if there is no "cntRows".
  • "maxRows" - sets the number of lines that, when multiplied by the line height, determine the max height. Active if there is no "cntRows".

These examples used the attribute: cntRows="2" or [cntRows]="cntRowsStr" or [cntRows]=" ' '+cntRows".
