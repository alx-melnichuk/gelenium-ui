Palette.

You can customize the color palette for the input element. To avoid duplicating color codes, we will use the following CSS variables: 

// Let's create our own color palette.
.fip-specific .Grn-palette {
  // The default border color.
  --fip-default-color: #00acc1;
  // Border and label color on error.
  --fip-error-color: #ad1457;
  // Border and label color on error and mouse hover.
  --fip-error-hover-color: #f06292;
  // The color of the frame and label when receiving focus.
  --fip-focused-color: #0097a7;

  // The default color for the label.
  --fip-label-default-color: #5c6bc0;
  // The color of the label if the element is disabled.
  --fip-label-disabled-color: #9c27b0;

  // For the exterior "Outlined":
  // The default border color.
  --fip-outlined-default-brd-color: #5c6bc0;

  // For the exterior "Underline":
  // Default background color.
  --fip-underline-default-bg-color: rgba(178, 235, 242, 0.6);
  // The background color when receiving focus.
  --fip-underline-focused-bg-color: rgba(178, 235, 242, 0.4);
  // The background color when the element is disabled.
  --fip-underline-disabled-bg-color: rgba(178, 235, 242, 0.2);

  // For the exterior "Underline" and "Standard":
  // The default bottom border color.
  --fip-substrate-before-brd-color: rgba(101, 1, 151, 0.9);
}

All the CSS classes that define the colors of the input element are children of the "Grn-palette" CSS classes. Add a custom "fip-specific" CSS class to define our colors.

To replace the color of the label, we define a group of CSS classes. 

// Default color of the label.
.fip-specific .Grn-palette .p-lbl-def-cl
{ color: var(--fip-label-default-color); }
// The color of the label on error.
.fip-specific .Grn-palette .p-lbl-err-cl
{ color: var(--fip-error-color); }
// The color of the label on mouse over and on error.
.fip-specific .Grn-palette .p-lbl-hov-err-cl
{ color: var(--fip-error-hover-color); }
// The color of the label on mouse hover.
.fip-specific .Grn-palette .p-lbl-hov-cl
{ color: var(--fip-default-color); }
// The color of the label when the input element receives focus.
.fip-specific .Grn-palette .p-lbl-foc-cl
{ color: var(--fip-focused-color); }
// The color of the label when the input element is disabled.
.fip-specific .Grn-palette .p-lbl-dis-cl
{ color: var(--fip-label-disabled-color); }

Likewise to replace the "Outlined" border color define a group of CSS classes. 

.fip-specific .Grn-palette .p-o-XXX-br-cl
{ border-color: var(--fip-outlined-default-brd-color); }

Where XXX - take the following values:

  • def - The default border color.
  • err - The border color in an error state.
  • hov-err - The border color on mouse hover in an error state.
  • hov - The border color on mouse hover
  • foc - The border color when getting focus.
  • dis - The border color in state is disabled.

Likewise, to replace the "Underline" background color, define a group of CSS classes. 

.fip-specific .Grn-palette .p-u-XXX-bg-cl
{ background-color: var(--fip-underline-default-bg-color); }

Where XXX - take the values from the previous example.

Likewise, to replace the "Underline" and "Standard" colors, define a group of CSS classes. 

.fip-specific .Grn-palette .p-su-XXX-br-cl:before 
{ border-bottom-color: var(--fip-substrate-before-brd-color); }

Where XXX - take the values from the previous example.
