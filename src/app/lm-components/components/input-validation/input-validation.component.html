The "isError" parameter sets the field to the error display state. The "helperText" parameter can be used to display an error message. A thin border (exterOutlined) or a red underline (exterUnderline | exterStandard) signals the user about an erroneous value in this input field. The red frame is displayed even if the input field is disabled (isDisabled).
