Palette customization

You can customize the color palette for the input element. To avoid duplicating color codes, we will use the following CSS variables: 

// Let's create our own color palette.
.ipc-customization .Grn-palette {
// The default border color.
--in07-default-color: #00acc1;
// Border and label color on error.
--in07-error-color: #ad1457;
// Border and label color on error and mouse hover.
--in07-error-hover-color: #f06292;
// The color of the frame and label when receiving focus.
--in07-focused-color: #0097a7;

// The default color for the label.
--in07-label-default-color: #5c6bc0;
// The color of the label if the element is disabled.
--in07-label-disabled-color: #9c27b0;

// For the exterior "Outlined":
// The default border color.
--in07-outlined-default-brd-color: #5c6bc0;

// For the exterior "Underline":
// Default background color.
--in07-underline-default-bg-color: rgba(178, 235, 242, 0.6);
// The background color when receiving focus.
--in07-underline-focused-bg-color: rgba(178, 235, 242, 0.4);
// The background color when the element is disabled.
--in07-underline-disabled-bg-color: rgba(178, 235, 242, 0.2);

// For the exterior "Underline" and "Standard":
// The default bottom border color.
--in07-substrate-before-brd-color: rgba(101, 1, 151, 0.9);
}

All the CSS classes that define the colors of the input element are children of the "Grn-palette" CSS classes. Add a custom "ipc-customization" CSS class to define our colors.

To replace the color of the label, we define a group of CSS classes. 

// Default color of the label.
.ipc-customization .Grn-palette .plt-clr-default
{ color: var(--in07-label-default-color); }
// The color of the label on error.
.ipc-customization .Grn-palette .plt-clr-error
{ color: var(--in07-error-color); }
// The color of the label on mouse over and on error.
.ipc-customization .Grn-palette .plt-clr-mouse-error
{ color: var(--in07-error-hover-color); }
// The color of the label on mouse hover.
.ipc-customization .Grn-palette .plt-clr-mouse
{ color: var(--in07-default-color); }
// The color of the label when the input element receives focus.
.ipc-customization .Grn-palette .plt-clr-focused
{ color: var(--in07-focused-color); }
// The color of the label when the input element is disabled.
.ipc-customization .Grn-palette .plt-clr-disabled
{ color: var(--in07-label-disabled-color); }

Likewise to replace the "Outlined" border color define a group of CSS classes. 

.ipc-customization .Grn-palette .plt-brd-clr-XXX
{ border-color: var(--in07-outlined-default-brd-color); }

Where XXX - take the values from the previous example.

Likewise, to replace the "Underline" background color, define a group of CSS classes. 

.ipc-customization .Grn-palette .plt-bg-clr-XXX
{ background-color: var(--in07-underline-default-bg-color); }

Where XXX - take the values from the previous example.

Likewise, to replace the "Underline" and "Standard" colors, define a group of CSS classes. 

.ipc-customization .Grn-palette .plt-brd-bt-clr-XXX:before 
{ border-bottom-color: var(--in07-substrate-before-brd-color); }

Where XXX - take the values from the previous example.

import { Component, ViewEncapsulation } from '@angular/core';
import { FormControl, FormGroup, Validators } from '@angular/forms';

@Component({
  selector: 'app-input',
  templateUrl: './input.component.html',
  styleUrls: ['./input.component.scss'],
  encapsulation: ViewEncapsulation.None,
})
export class InputComponent {
  public minLength07e = 5;
  public maxLength07e = 15;
  public controls07 = {
    input07a: new FormControl('', []),
    input07b: new FormControl('Hello World', [Validators.required]),
    input07c: new FormControl('Hello World', []),
    input07d: new FormControl('Hello World', []),
    input07e: new FormControl('demo', [
      Validators.required,
      Validators.minLength(this.minLength07e),
      Validators.maxLength(this.maxLength07e),
    ]),
    input07f: new FormControl('Hello World', []),
  };
  public formGroup07: FormGroup = new FormGroup(this.controls07);
  public exterior07 = 'outlined';
}
