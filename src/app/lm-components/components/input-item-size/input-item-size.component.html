Item size.

You can change the height of the element. This requires adding one of the following attributes:

  • "szShort" - sets the height to 38px;
  • "szSmall" - sets the height to 44px;
  • "szMiddle" - sets the height to 50px;
  • "szWide" - sets the height to 56px (default);
  • "szLarge" - sets the height to 62px;
  • "szHuge" - sets the height to 68px;

The size of the element's height is set in the "‑‑gfi‑base-size" ccc variable in accordance with the "szXXX" size attribute. All padding and offset calculations are determined by the value of this variable and the font size.

In order to hide the label (legend), add the "hiddenLabel" attribute.

The font size for native input and label is determined from the parent container. The size of the font can be defined with the css-style for the "grn-input" tag. In the example, this is implemented with the styles "iis-fn-sz-large" and "iis-fn-sz-huge". We can change the font size at the grn-input tag level, but the element itself will not change the height.

The height of the element remains the same even when the font size is changed (from font-size: 10px to font-size: 26px).
