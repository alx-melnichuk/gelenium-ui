Item size.

The input element has the ability to change the height. This requires adding one of the following attributes:

  • "szShort" - sets the height to 38px;
  • "szSmall" - sets the height to 44px;
  • "szMiddle" - sets the height to 50px;
  • "szWide" - sets the height to 56px (default);
  • "szLarge" - sets the height to 62px;
  • "szHuge" - sets the height to 68px;

It is possible to hide the display of the label (legend). In this case, for the "Standard" mode, the height of the element is reduced by a value of 1em.
