Ornaments for the input line.

It is possible to add a different ornament (picture or text) for the input line. It will appear as a prefix or suffix for the input string. You can define the reaction to pressing an element of the ornament.
To add a prefix, you need to add a tag with the "grnOrnamentLf" attribute inside the "grn-input" tag. (<span grnOrnamentLf>kg</span>)
To add a suffix, you need to add a tag with the "grnOrnamentRg" attribute inside the "grn-input" tag. (<span grnOrnamentRg>cm</span>)
Adding a prefix or suffix turns off the animation of the legend for this input field.

For example:

  • you can use the prefix "kg" to enter the weight value;
  • you can use the "cm" suffix to enter the height value;
  • you can use the button with the icon to display the password;

You can increase/decrease the font size for an input element without changing the size of the element. For example, when entering a password, you can increase the font size to display larger black circles.
To do this, add a style: 

.io-ornament .io-passwd-big .gi-input {
  font-size: 20px;
}

You can add the "lbShrink" attribute and the legend will be fixed in the title. Nor will the legend descend to the pre-holder position.
